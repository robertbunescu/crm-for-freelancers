# Freelancer CRM Dashboard

A modern SaaS-style CRM built for freelancers to manage leads, sales pipeline, clients, and tasks — all in one place.

Designed with a focus on clean UI, smooth UX, and scalable component architecture.

---

## ✨ Features

* 📊 **Dashboard** — revenue overview, lead sources, activity tracking
* 🧲 **Leads Management** — track and manage incoming leads
* 🔄 **Pipeline** — visual deal tracking with drag-and-drop stages
* 👥 **Clients** — manage active clients and revenue
* ✅ **Tasks** — organize work and track progress
* 🌙 **Dark Mode** — seamless theme switching
* ⚡ **Modern UI/UX** — inspired by Linear, Notion, and Stripe

---

## 🛠 Tech Stack

* **Next.js 16 (App Router)** + **React 19** + **TypeScript**
* **Tailwind CSS** + **shadcn/ui** (Radix UI primitives)
* **Recharts** for data visualization
* **react-hook-form** + **sonner** for forms & notifications
* **lucide-react** for icons
* **next-themes** for dark mode
* **date-fns** for date handling

---

## 🧠 Architecture

* App Router with route groups for authenticated sections
* Component-driven UI using reusable shadcn patterns
* Centralized design system (spacing, colors, typography)
* Mock data layer (no backend, frontend-focused project)

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.
